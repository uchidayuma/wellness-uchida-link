最終更新: 2026-05-08

## 基本方針

**「自分はこういう人間です」から入る。シンプルに。**

- 課題提起（こんな困りごとはありませんか？）はサービス個別ページに書く
- トップは「誰か・何ができるか・実績」を順番に見せるだけ
- 読み終わったら問い合わせたくなる流れを作る

## 各セクション詳細

### 1. About / Hero
- キャッチコピー: `コンテナ環境のセキュリティと構築を、任せられる専門家。`
- サブ: `Docker / AWS ECS / Kubernetes の実務経験をもとに、セキュリティ診断からインフラ構築まで対応します。`
- 実績バッジ: Udemy 2.6万人・Best Seller・YouTube 6,000人
- プロフィール写真
- CTA（主）: **「まずは相談する」**（Contact へスクロール）
- CTA（副）: レジュメをダウンロード / GitHub

### 2. Services
- 3枚のカード
  1. コンテナ・ソースコードセキュリティ診断（¥300,000〜）→ 詳細ページへ
  2. ECS / Kubernetes インフラ構築支援（要相談）→ 詳細ページへ（新規作成）
  3. スポット相談 60分（¥5,000 税抜）→ Contact へ

### 3. なぜ任せられるか
- Udemy Best Seller / 2.6万人受講生
- YouTube 6,000人登録者・5,000再生超
- インプレス研修 60名・満足度95%+
- 本番 ECS/Fargate 移行・CI/CD 構築実績

### 4. Works（3件）
- 宿泊施設向け予約管理・オートロック管理ツール（CDK・IaC・多国籍チーム）
- 医療機関向けCRM / LINEミニアプリ（20名PM・バグ発生率半減）
- フランチャイズ向け基幹システム（AWS移行・CI/CD・Laravel 6→10）

### 5. Career
- 現行の略歴4行をそのまま使用（変更なし）

### 6. Skills
- 現行のまま（変更なし）

### 7. Contact
- 問い合わせフォーム
- クイック導線（LINE・カレンダー）
